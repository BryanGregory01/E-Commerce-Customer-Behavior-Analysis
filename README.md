# E-Commerce-Customer-Behavior-Analysis

Tools:
- PostgreSQL
- SQL

What I did:
- Built relational database with 6 tables
- Created realistic dataset (50+ customers, 60 orders, 120+ items)
- Analyzed revenue, customer behavior, product performance, and delivery metrics

Skills:
- JOIN
- GROUP BY
- Aggregations
- Window functions
- Data analysis

Insights:
- Top customers generated highest revenue
- Electronics category drove most sales
- Certain regions showed higher demand
- Delivery performance varied by carrier
